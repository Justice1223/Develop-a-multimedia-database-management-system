# 🎬 Multimedia Database Management System (Netflix-like)

## 📌 Project Overview
This project implements a **multimedia database management system** similar to Netflix. It is designed to manage content, users, subscriptions, and interactions using a fully normalized relational database in MySQL.

The system supports:
- Content management (movies, TV shows)
- User subscriptions and payments
- Watchlists and viewing history
- Reviews and ratings
- Automated business logic via triggers, procedures, and events

---

## 🛠️ Tech Stack
- **MySQL 8.0** – Database management system
- **MySQL Workbench** – Database interface
- **Python 3** – Data ingestion script
- **SQL** – Schema, queries, triggers, procedures
- **Draw.io** – ER Diagram design
